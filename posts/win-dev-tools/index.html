Everything (Realtime filesystem indexing)

To begin let’s start by downloding the tool from this link Downlaod link

Introduction

Everything it’s a tool that query NTFS filesystem for any change that happen to any referenced device and provide a way to have control over you machine. the binary it self is no more than 2 megs. and it can change the way you interact with windows explorer forever.

If you combine the tool with a hotkey you can decrease the use windows explorer to 10%

Features

  • Advanced Realtime search
  • Remote Realtime search via mini Http server
  • Remote Realtime search via mini Ftp server

Use Cases

  • search a specefic file
  • search a pattern ‘*.dll, *.exe, …’
  • search a pattern under a location
  • search the count of pattern
  • search under a specefic location

example: you want to search for the host file under system

type in the search field : \system hosts

if you want to filter more add !*.ics !*.sam and you will have only your file.

The final result should be \system hosts !*.ics !*.sam.
